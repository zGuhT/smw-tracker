Game Setup

How This Works ▾

This page lets you define the levels and runs for a game. These definitions are shared across all users — if someone else has already set up a game, you can import their config from the Community Configs section below.

Level Definitions

Each level needs a name and a Level ID (a 2-character hex code). The Level ID is the internal address the SNES uses for the level — the tracker reads this from hardware memory to know which level you're in. If a level has a secret exit (keyhole), check the "Secret exit" box — the tracker will detect keyhole activations and record it as a separate split.

Tip: If you're on your local machine with QUsb2Snes connected, you can click "Capture" while standing in a level to auto-fill the Level ID from hardware. On the web, you'll need to enter it manually — you can find level IDs in ROM maps or Lunar Magic's level list.

Run Definitions

A run is an ordered list of levels that defines a speedrun category (e.g. "100%", "Any%", "All Exits"). The tracker uses the run definition to:

  • Show splits in the correct order on the live page
  • Track your PB (personal best) and sum of best
  • Compare your current run against your best

Set the start delay (in milliseconds) to account for the time between selecting a file and gaining control (e.g. 3200ms for most SMW hacks). Mark one run as Default so the tracker auto-selects it when you start playing.

Use "Add All Levels" to populate the run with every defined level, then remove or reorder as needed. Drag the ≡ handles to change the order.

Sharing

Once you're happy with your setup, click "Publish to Community" to share it. Other users can then import your config with one click. You can also Export your config as a JSON file for backup.
