Download the Tracker

The SMW Tracker client runs on your PC, connects to your SNES hardware via QUsb2Snes, and automatically tracks your splits, deaths, and progress in real-time. Your data syncs to the cloud so anyone can watch your runs live.

What You Need
  • Python 3.10+ — python.org/downloads
  • QUsb2Snes — connects to SD2SNES/FXPak Pro, or compatible emulator
  • An SMW Tracker account — sign up here (free)
Installation
# 1. Clone the tracker client
git clone https://github.com/zGuhT/smw-tracker.git
cd smw-tracker

# 2. Install Python dependencies
pip install -r requirements.txt

# 3. Make sure QUsb2Snes is running and your SNES is connected

# 4. Start the tracker with your API key
#    (find your key at smwtracker.com → Account)
python run_tracker.py --cloud --api-key YOUR_API_KEY
How It Works
  1. The tracker reads your SNES memory via QUsb2Snes (4 times per second)
  2. It detects game state: which level you're in, deaths, exits, keyhole activations
  3. Splits are recorded automatically when you complete a level
  4. Session data syncs to smwtracker.com so viewers can watch live
  5. Your stats, PBs, and split history are saved to your profile
A Note About the Code

The tracker client and the website share the same repository. The code is open source so you can see exactly what it does — no secrets are stored in the repo. Your API key, passwords, and email are never in the code; they're stored securely on the server and in environment variables. The tracker client only needs your API key to authenticate, and it only sends game state data (levels, splits, deaths) to the cloud.

A standalone desktop client with a GUI is planned for the future. For now, the Python script is the way to go.
