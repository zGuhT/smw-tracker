Download the Tracker

The SMW Tracker client connects to your SNES hardware via QUsb2Snes and automatically tracks your splits, deaths, and progress in real-time.

Requirements
  • Python 3.10+
  • QUsb2Snes connected to your SNES (SD2SNES/FXPak Pro, or emulator)
  • An SMW Tracker account (sign up here)
Quick Start
# Clone or download the tracker
git clone https://github.com/your-repo/sfc-tracker.git
cd sfc-tracker

# Install dependencies
pip install -r requirements.txt

# Start the tracker (use your API key from account settings)
python run_tracker.py --cloud --api-key YOUR_API_KEY

A standalone desktop client is coming soon. For now, use the Python script above.
